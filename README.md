# UrbanWalks

UrbanWalks is a university web development project created to promote enjoyable city walking experiences. 
Unlike traditional navigation tools that focus only on efficiency, UrbanWalks highlights scenic, architecturally interesting, and peaceful walking routes in Budapest.

The website includes:
- an About Us homepage,
- curated recommended walking routes,
- an interactive image gallery,
- and a user street rating form.

Built with HTML5, CSS3, and vanilla JavaScript.
